使用说明

一、使用案例

  • 1. 关注股票实时价格

    • 1.1 浏览器中复制股票的网址，粘贴网页设置界面中网址栏；

    • 1.2 选择触发条件是数字比较，当网页加载完毕时候，点击所关注的数字，绿框表示可以被监测，否则红框表示不可以被监测；

    • 1.3 选择比较符号和输入期望的数字；

    • 1.4 保存并运行之后，应用会每隔一定频率监测网页，与目标数字进行比较，如果符合触发条件，应用将会响应。

  • 2. 关注微博热搜、百度热榜等热点网址的关键信息

    • 2.1 浏览器中复制热点网址，粘贴网页设置界面中网址栏；

    • 2.2 选择触发条件是文本比较-当关键字显示；

    • 2.3 输入（多个）关键字。例如关注社会事件、明星等关键字；

    • 2.4 保存并运行之后，应用会每隔一定频率监测网页，如果网页中出现关键字，应用将会响应。

  • 3. 关注你所需了解最新网页信息

    • 3.1 浏览器中复制需要实时关注的网站（例如招聘、公司公告、旅游、移民等官网最新消息），粘贴网页设置界面中网址栏；

    • 3.2 选择触发条件是文本比较-当任何文本改变；

    • 3.3 注意的是等待整个网页加载完毕，才可以进行保存；

    • 3.4 保存并运行之后，应用会每隔一定频率监测网页，监测网页任何文本是否发生变化，如果与之前保存网页信息不一样，应用将会响应。

  • 以上，当满足触发条件，应用就会响应（手机就会发出铃声、振动、闪灯或本地通知）。

二、其他问题

  • 1. 网页中哪些数据不能监测？

    • 1.1 网页中图片、图表（canvas）、图标、flash player控件等无法监测其中文本或数据内容。

    • 1.2 若监测的文本、数据需要点击或切换按钮加载而来，是无法监测的。注意的是当触发条件是监测整个网页变化时候，不要点击或切换任何需要加载数据的按钮。

    • 1.3 若需要用户登录之后，网页信息加载文本或数据，是无法监测。

  • 2. 当手机被锁屏，应用将不能进行监测。

    因为手机被锁屏，苹果手机系统会暂停正在运行中应用程序。所以不要手动锁屏和开启从不锁屏，保持本应用前台且正常运行中。如果为了降低手机用电消耗，可以开启省电模式，把手机翻转，手机屏幕将会变暗。

  • 3. 当应用处于后台时候，应用程序也不能进行监测。

    因为应用被置于后台，苹果手机系统为了释放更多内存，会暂停处于后台的应用程序。所以请将本应用放置前台，并保持正常运行。

  • 4. 如何取消会员自动续订？

    4.1 首先，购买连续包周/月/年会员，会在到期前24小时，自动续订并扣除相应费用。

    4.2 如需取消自动续订，请至少提前24小时操作取消。您可通过下述方式中止或终止本服务：苹果应用内支付（IAP支付，in-app purchase）自动续费用户退订方法如下： 打开苹果手机“设置” --> 进入“iTunes Store 与 App Store”-->点击 “Apple ID”，选择"查看Apple ID"，进入"账户设置"页面，点击“订阅”，选择“连续包周/月/年会员”取消订阅即可。

    4.3 同一个Apple ID只能给一个「网页雷达」账号开通自动续费服务。
